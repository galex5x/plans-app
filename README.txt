Планы (PWA, офлайн)

Как установить на iPhone:
1) Размести папку на любом HTTPS-хостинге (GitHub Pages / Netlify / Vercel / Cloudflare Pages).
2) Открой ссылку в Safari на iPhone.
3) Кнопка «Поделиться» → «На экран Домой» (Add to Home Screen).
4) Запусти с иконки. После первого открытия приложение работает офлайн.

Важно:
- Данные хранятся локально на телефоне (localStorage).
- Сделай экспорт в JSON на всякий случай (Настройки → Скачать экспорт).
